## useClientEffects$()

- useClientEffects$() executes code after the component is resumed. The code will only run in the client side.
- useClientEffects$() get a callback function and will run when the component resumes from the pause point from the server.
- You need to return a cleanup function so when the component is destroyed the cleanup function will be executed to cleanup all resource initialized in the useClientEffects$() hook.
